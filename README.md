## Overview

This repository contains the supplementary materials for the paper **“An Empirical Investigation of Ethical Concerns in Online Survey Research in Software Engineering”**.
The materials include the full questionnaire used in the study, anonymized responses from the online survey, and the annotation results of the literature sample analyzed in the research.

All files are provided to support transparency, reproducibility, and further research on ethical considerations in online survey research within software engineering.

---

## Repository Contents

### 1. **questionnaire.pdf**

This file contains the complete English version of the questionnaire used in our study.
The questionnaire includes sections on participants’ experiences with survey-based research, ethical considerations, recruitment practices, and perceived challenges.

---

### 2. **results_questionnaire.xlsx**

This spreadsheet provides the responses of the 54 valid participants who completed the online survey.

* All personally identifiable information (PII) has been removed.
* Only aggregated or non-identifiable attributes (e.g., career stage, research experience) are retained.
* The dataset is provided solely for research and replication purposes.

---

### 3. **results_literature.xlsx**

This file contains the annotation results for the 340 papers included in our systematic literature review.

For each paper, the file reports:

* Basic bibliographic information
* Identified recruitment methods
* Reported ethical issues and practices
* Any additional coded attributes used in the analysis

This dataset reflects the coding decisions described in the paper and can be used for validation or further secondary analysis.

---

## Ethical and Privacy Considerations

All data contained in this repository have been fully anonymized.
No personal information about survey respondents or paper authors is included.
The release of these materials follows the ethical guidelines approved by our institutional ethical review committee.
